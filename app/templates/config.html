Manual Update Commands

Run from this repository root on your Docker host to update a local/self-hosted deployment and keep Installed Commit aligned with the running build. 

git pull --ff-only
HEV_BUILD_COMMIT_SHA=$(git rev-parse HEAD) docker compose up -d --build --force-recreate
curl -fsS http://localhost:23010/version

If your deployment uses a non-default host port, replace 23010 in the curl commands.
